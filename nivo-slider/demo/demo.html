参数说明:

  1. effect: 设置幻灯变化效果，可以使用'random', 或者指定效果如, sliceDownRight','sliceDownLeft','sliceUpRight','sliceUpLeft','sliceUpDown','sliceUpDownLeft','fold','fade', 'boxRandom','boxRain','boxRainReverse','boxRainGrow','boxRainGrowReverse'
  2. slices: slice动画效果参数
  3. boxCols: box动画效果参数
  4. boxRows:box动画效果参数
  5. animSpeed: 动画变化速度
  6. pauseTime: 每个幻灯时间
  7. startSlide: 起始幻灯
  8. directionNav: 是否显示“上一个”和“下一个”导航
  9. controlNav: 是否显示导航控制
  10. controlNavThumbs: 导航是否使用缩略图
  11. pauseOnHover: 悬浮是否停止
  12. manualAdvance: 是否手动变化幻灯
  13. prevText: 上一张文字设定
  14. nextText: 下一张文字设定
  15. randomStart: 是否随机开始
  16. beforeChange:function(){} 函数，在幻灯变化前调用
  17. afterChange:function(){}函数，在幻灯变化后调用
  18. slideshowEnd:function(){}函数，在幻灯变化完成调用
  19. lastSlide:function(){}函数，在最后一个幻灯变化完成调用
  20. afterLoad:function(){} 函数，slider加载后调用
 
    $('#slider').nivoSlider({
        effect: 'random',                 // Specify sets like: 'fold,fade,sliceDown'
        slices: 15,                     // For slice animations
        boxCols: 8,                     // For box animations
        boxRows: 4,                     // For box animations
        animSpeed: 500,                 // Slide transition speed
        pauseTime: 3000,                 // How long each slide will show
        startSlide: 0,                     // Set starting Slide (0 index)
        directionNav: true,             // Next & Prev navigation
        controlNav: true,                 // 1,2,3... navigation
        controlNavThumbs: false,         // Use thumbnails for Control Nav
        pauseOnHover: true,             // Stop animation while hovering
        manualAdvance: false,             // Force manual transitions
        prevText: 'Prev',                 // Prev directionNav text
        nextText: 'Next',                 // Next directionNav text
        randomStart: false,             // Start on a random slide
        beforeChange: function(){},     // Triggers before a slide transition
        afterChange: function(){},         // Triggers after a slide transition
        slideshowEnd: function(){},     // Triggers after all slides have been shown
        lastSlide: function(){},         // Triggers when last slide is shown
        afterLoad: function(){}         // Triggers when slider has loaded
    });
